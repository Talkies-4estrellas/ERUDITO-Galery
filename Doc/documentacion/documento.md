# ERUDITO Galery — Documentación Técnica

## Stack
- **Framework:** Next.js 16 (App Router)
- **Estilos:** Tailwind CSS v4
- **Base de datos:** Supabase PostgreSQL
- **Auth:** Supabase Auth (`signInWithPassword`, `signUp`, `onAuthStateChange`)
- **Storage:** Supabase Storage — bucket `obras` (público)
- **Deploy:** Vercel — rama `master`
- **Lenguaje:** TypeScript

## Base de datos — Tablas principales

### `artistas`
```sql
id_artista  serial PK (generated by default)
nombre      text NOT NULL
vida        text           -- ej: "1890 – 1960"
origen      text
especialidad text
foto_perfil text           -- URL
biografia   text
email       text           -- nullable (DROP NOT NULL migrado)
```

### `obras`
```sql
id_obra         serial PK (generated by default)
titulo          text NOT NULL
anio            text
precio          int
id_artista      int FK → artistas
imagen_url      text
tipo            text           -- Pintura, Fotografía, Escultura, etc.
estrellas       int DEFAULT 5
grafica_valor   jsonb
grafica_interes jsonb
certificaciones jsonb
-- (más columnas según schema.sql)
```

### `eventos`
```sql
id_evento   serial PK (generated by default)
titulo      text NOT NULL
lugar       text
modalidad   text CHECK (Presencial|Virtual|Híbrido|En línea)
descripcion text
imagen_url  text
fecha_corta jsonb    -- { "mes": "ABR", "dia": "12" }
href        text     -- URL del evento externo
fichas_ids  jsonb    -- array de id_obra relacionadas
```

### `perfiles`
```sql
id          uuid PK FK → auth.users
rol         text CHECK (artista|comprador|empresa|admin) NOT NULL
nombre      text
bio         text
especialidad text
pais        text
slug        text    -- solo rol empresa
avatar_url  text DEFAULT ''
```

| Rol | Descripción | Panel propio |
|---|---|---|

## Flujo de autenticación

```
/registro → elige correo/contraseña → Supabase signUp
    ↓
/registro?paso=rol → SelectorRol → usePerfil.elegirRol()
    ↓
/perfil → PaginaPerfil → (MiPerfilArtista | MiPerfilEmpresa | PerfilComprador)
```

- `useAuth` — suscripción a `onAuthStateChange` → expone `{ user, cargando, salir }`
- `usePerfil` — lee/escribe tabla `perfiles` → expone `{ perfil, listo, elegirRol, guardar, cerrarSesion }`
- `AuthGuard` — si no hay sesión redirige a `/login`

## Subida de imágenes (Storage)

**Endpoint:** `POST /api/upload?carpeta=avatars`

**Flujo:**
1. Cliente selecciona imagen
2. `uploadWebp(file, url)` — convierte a WebP vía Canvas API
3. POST al endpoint con `FormData`
4. Supabase Storage guarda en bucket `obras` → subcarpeta según `carpeta` param
5. Retorna URL pública

**Subcarpetas en uso:**
- `obras/` — imágenes de obras subidas por artistas/empresas
- `avatars/` — fotos de perfil y logos
